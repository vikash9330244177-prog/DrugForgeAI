Advanced 3D GNN Screening

This module is designed for conformer-based ligand screening. It will use RDKit 3D conformer generation, geometry optimization, distance-aware molecular graphs, EGNN/SchNet-style message passing, attention pooling, uncertainty-aware prediction, applicability-domain checking, and 3D priority scoring.

Recommended input:
  • Training CSV: must contain SMILES and Activity columns.
  • Prediction CSV: must contain a SMILES column.
  • 3D conformer generation is slower than 2D GNN screening but can capture geometry-dependent molecular patterns.

Welcome to DrugForge AI – Your Automated Virtual Screening Companion

DrugForge AI leverages deep learning to streamline computational drug discovery, making the process faster, more accurate, and more accessible. Whether you are refining protein structures, prioritizing ligands, performing molecular docking, evaluating ADMET properties, building QSAR models, optimizing hits, or analyzing protein–ligand interactions, DrugForge AI is designed to support each stage of the workflow.

Getting Started:

  • Upload Your Data: Begin by uploading your protein files and ligand datasets. DrugForge AI accepts PDB files for proteins and CSV files for ligand screening. Ensure that ligand files contain the required columns such as SMILES and Activity where applicable.
  • Enhanced 2D GNN Screening: Use the improved graph neural network model with rich atom/bond features, attention pooling, uncertainty scoring, and applicability-domain checking to rank ligands.
  • Advanced 3D GNN Screening: Use conformer-based molecular geometry and distance-aware graph learning for more accurate ligand prioritization when 3D molecular shape is important.
  • Protein Refinement: Upload your protein structure for refinement in order to improve the quality of downstream docking analysis.
  • Molecular Docking: Use the Molecular Docking tab to perform focused docking with manually specified search box coordinates and dimensions.
  • Blind Docking: Use the Blind Docking tab to perform whole-protein exploration and multi-site docking analysis.
  • ADMET Prediction: Use the ADMET tab to evaluate pharmacokinetic and toxicity-related properties for single molecules or uploaded SMILES libraries.
  • QSAR Modeling: Use the QSAR tab to train predictive models and run external property or activity prediction on new molecular libraries.
  • Hit-to-Lead Optimization: Use the Hit-to-Lead tab to refine prioritized hits, filter analogs, and identify stronger lead candidates.
  • Protein–Ligand Interaction Analysis: Use the interaction analysis tab to inspect residue-level contacts and summarize binding patterns for docked or experimental complexes.
  • Analysis and Download: Review the generated results directly within DrugForge AI and download the outputs for further analysis.

Tips for Success:

Ensure that all input files are correctly formatted and contain the necessary information. The de novo molecule generation, QSAR, ADMET, hit-to-lead, interaction analysis, and re-screening modules can be combined to iteratively expand and refine the compound selection workflow.

Technical Support:

If you encounter issues, first verify file formatting and dependency installation. Re-run the workflow only after confirming that all required libraries, Python environments, and executables are properly configured.

Disclaimer:

DrugForge AI is intended for research use only. Computational predictions should be interpreted with caution and validated experimentally whenever possible.
